A.Suitable type requirements

  1. a.Android system to be 4.3 or higher;
  2. b.IOS system to be 6.0 or higher(at least for Iphone 4s);
  3. c.Bluetooth version of the phone to be 4.0 or higher.

B.What to do if the Bluetooth signal cannot be searched out?

  1. a.Make sure Bluetooth is turned on;
  2. b.Make sure that the vehicle brand has been selected and that the parameters of the vehicle have been selected
  3. c.Make sure the vehicle is not connected.

C.Unique identification

  1. a.Unique identification is a necessary process for the connection between APP and the vehicle;
  2. b.Click the "BX - *****" Bluetooth signal, enter the authorization page, press the“eMaster”key and “lock” key on the remote at the same time to complete the identification.

D.Custom function on the remote

  1. a.The function of setting the "eMaster" key is required when the vehicle connection is successful;
  2. b.Only one of the driving modes can be selected for the basic mode of "eMaster" key.

E.Using the "eMaster" key on the remote

  1. a.The default custom function is "Over-speed" mode, it could be changed according to your need;
  2. b.Before using, keep the vehicle speed at "zero" state;
  3. c.Restart the ignition lock after each time using,then it will enter“normal riding”mode.

F.Switching between driving modes

  1. If you want to switch to another travel mode while the current travel mode is in the "smart parking" or "intelligent pushing" mode, please exit the current travel mode according to the prompt first.

G.Safety Mode

  1. In the process of riding, turning the throttle, but the speed cannot be improved, you may enter the “speed limit”state, please turn the "safety mode" button off (slip to the left).

H.Why can not display horizontal screen after rotating mobile phone?

  1. Horizontal screen display function could be used when your phone is in rotating-screen mode only,make sure that the phone screen is unlocked.

二、适配机型要求

  1. a.Android系统： 4.3版本以上；
  2. b.IOS系统：6.0版本以上（即：4S以上）；
  3. c.手机蓝牙版本需4.0以上。

三、搜索不到蓝牙信号怎么办

  1. a.确认手机蓝牙处于打开状态；
  2. b.确认已选择车辆品牌，并已选择车辆的各项参数；
  3. c.确认车辆不在已连接状态。
  4. d.在手机“设置”中寻找到帮大师软件，释放其在手机中的权限设置。

四、行驶模式间的切换

  1. 当前行驶模式处于“智能泊车”或“智能推行”状态时，若想切换成其他的行驶模式，请先根据提示退出当前行驶模式。

五、安全模式

  1. a.在行驶过程中若遇转动转把速度无法提升时，你可能进入了限速行驶状态，请将“安全模式”的按钮关闭（将按钮划至左侧）。
  2. b.尽量不要在“超速”行驶状态下，使用安全模式；
